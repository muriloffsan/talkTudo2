## Como Executar o Projeto

1. Clone este repositório:
	```bash
	git clone https://github.com/seu-usuario/talkTudo2.git
	```
2. Navegue até o diretório do projeto:
	```bash
	cd talkTudo2
	```
3. Instale as dependências:
	```bash
	npm install
	```
4. Inicie o servidor:
	```bash
	npm start
	```
5. Acesse o site no navegador:
	```
	http://localhost:3000
	```
